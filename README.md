- **Styling:** Tailwind CSS 4
- **Components:** Radix UI

## 📦 Installation

```bash
# Clone the repository
git clone https://github.com/ziademad02153/personal-web.git

# Navigate to directory
cd personal-web

# Install dependencies
npm install --legacy-peer-deps

# Run development server
npm run dev
```
